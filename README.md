# dark-mode-toggle

## Learning Objectives
- Understand how to replicate a cool Dark Mode Toggle application which features a toggle button that adds different background themes on every toggle of the button to provide alternative backgrounds according to users taste a cool UI.
- Flexbox to place elements in the page.

## Description of the Project
For the first milestone in the process of creating Dark Mode Toggle, you will:
- set up a new repository and prepare it for development using best practices.
